Foundation

Foundation is a front-end framework. It comes with many different objects and classes that have already been designed, to make styling your websites much easier!

For example, you can turn basic links into buttons by labeling them with various classes.

Simple Button

You can create the above button from a normal link by adding the attribute class="button" inside the a tag.

Success Btn

This time, we add two classes, class="success button", to the link.

Alert Btn

This button uses class="alert button".

Secondary Btn

This button uses class="secondary button".
