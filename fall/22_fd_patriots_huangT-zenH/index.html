Getting Started

The button classes shown above, and all of the other classes that Foundation comes with, are stored in a CSS file that you can download. However, you can start using file right away by adding <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites@6.5.3/dist/css/foundation.min.css"> in the head of your HTML file. The CDN link we use leads to the compressed version of the full CSS file.

In addition, you can add your own CSS file by adding <link rel="stylesheet" href="css/app.css"> underneath your CDN link. Creating an app.css file allows you to write your own CSS to customize your page! It also allows you to edit any pre-made elements that you want to use. If you wanted to make the success button blue instead of green, you can copy the CSS code for that class into app.css and change the color!

Nav Bars

These pre-styled classes are very helpful when adding navigation bars to your website! One of the most useful features of Foundation, these nav bars save you from the frustration of moving links into the correct place.
